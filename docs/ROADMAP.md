# 开发路线

## Phase 1: 可运行骨架

- 用户注册/登录 API。
- 创建模拟任务。
- 设置初始模拟资金。
- 添加关注资产。
- 选择策略模板。
- mock 行情 provider。
- 模拟买入/卖出、交易日志、持仓和现金更新。
- 前端基础路由和页面骨架。

## Phase 2: 行情和策略

- 接入 Yahoo Finance 或 Finnhub。
- 实现均线交叉、RSI、DCA 三个策略模板。
- 支持策略参数配置。
- 支持冻结/恢复买入、冻结/恢复卖出。
- 支持追加/减少资金。
- 支持结束模拟任务。

## Phase 3: 分析能力

- 每日汇总：净值、现金、持仓市值、日收益、交易次数。
- 月度汇总：月收益、最大回撤、胜率、手续费税费。
- 交易日志筛选和导出。
- 任务对比页。

## Phase 4: 计算金融展示增强

- Sharpe / Sortino / Calmar。
- Benchmark 对比。
- 因子和信号解释。
- T+1、涨跌停、交易日历、期货合约乘数。
- 回测和准实时模拟的统一报告格式。
